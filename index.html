BlackJack (Create task)

See the Replit

My friend and I made the game BlackJack on Replit, an online coding platform, using Python. It has a simple design with just lines of text. We used multiple libraries for the deck of cards and to simulate the hand for both the player and the computer. We used functions for everything else like shuffling the deck, starting the game, and even giving the computer a basic AI so it would draw cards if the value of its hand is lower than 16.
