Godot Game Prototype Project

Building a prototype game to learn the game development process.

Background

In September of 2024, I started the game with 1 of my friends in the same Web APP class. It was for our first-semester final project. We had a choice between making a Website, phone app, or game. We chose to make a game because I thought it would be fun.

The only requirement for this project is to make a simple prototype of our game and we had 2 weeks to do it. We thought of other types of games but in the end, we settled on a top-down shooter type of game.
